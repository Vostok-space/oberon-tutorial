Повторите рассмотренный пример модуля, набрав его в редакторе. Если необходимо, можете раскрыть образец. Используйте кнопку My для проверки модуля в промежуточном состоянии, и кнопку My.Go для запуска процедуры в готовом модуле. Для ввода заглавных букв подряд удобно использовать клавишу Caps Lock.

образец 
MODULE My;

 IMPORT Out;

 PROCEDURE Go*;
 BEGIN
  Out.String("Привет всем!")
 END Go;

END My.

По аналогии добавьте в модуль вторую процедуру — Do, и запустите её с помощью соответствующей кнопки.
